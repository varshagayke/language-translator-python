# language-translator-python
# Language Translator using Python  This project translates text from one language to another using Python.

Features:
- User input for text
- Select source language
- Select destination language
- Real-time translation
- Error handling

Technologies used:
- Python
- googletrans library

Example:
Input: Hey it's me
From: English
To: Hindi

Output: अरे! यह मैं हूं
